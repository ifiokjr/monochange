# Add hosted release commit delegation

CommitRelease can now select a hosted backend that packages release file changes and sends them to monochange app using GitHub Actions OIDC or a monochange token. The schema and CLI input metadata expose the backend and auth options so release workflows can delegate verified release PR commits to the hosted GitHub App path.
