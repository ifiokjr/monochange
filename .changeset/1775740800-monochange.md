# preview release file diffs during dry-runs

`mc release` and other `PrepareRelease`-based commands now accept `--diff` so you can inspect version and changelog edits without mutating the workspace first.

Before:

```bash
mc release --dry-run --format text
mc release --dry-run --format json
```

After:

```bash
mc release --dry-run --diff
mc release --dry-run --format json --diff
```

Text output now prints unified file previews for planned manifest and changelog edits, with terminal colors when supported. JSON output keeps the normal manifest data under `manifest` and adds `fileDiffs` entries when `--diff` is requested.
