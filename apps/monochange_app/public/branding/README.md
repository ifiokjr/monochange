## Color palette

All logos use the monochange brand gradient:
- **Primary**: `#0c8ee7` (blue)
- **Accent**: `#7c3aed` (violet)
- **Secondary**: `#059669` (green)
- **Warm**: `#f59e0b` (amber)
- **Teal**: `#0891b2`

## My recommendations

- **App icon / favicon**: #1 (Interlocking packages) or #8 (Delta with M)
- **Wordmark / header**: #3 (Stylized M) with "monochange" text
- **Fork feature**: #7 (Fork and merge)
- **Community page**: #9 (Infinity packages)

## Better tools for logo generation

SVGs from code are a good starting point but for polished logos consider:

1. **Figma** (free) — vector design, export SVG, collaborative
2. **Penpot** (free, open-source) — Figma alternative
3. **Inkscape** (free, desktop) — traditional vector editor
4. **LogoFast** (free, web) — AI-generated logos, quick iterations
5. **Looka** (paid, web) — AI logo generator with brand kit
6. **SVGR** (free, web) — SVG optimization and cleanup
7. **RealFaviconGenerator** (free, web) — favicon generation from SVG
