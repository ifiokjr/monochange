# Tooling and commands

## Environment

- Enter the reproducible development shell with `devenv shell` before using repo task commands.
- Install workspace tooling with `install:all` when needed.

## Common commands

- `monochange --help`
- `mc --help`
- `mc validate`
- `build:all`
- `lint:all`
- `test:all`
- `coverage:all`
- `build:book`

## Useful task-specific commands

### Documentation

- `docs:check`
- `docs:update`
- `docs:verify`
- `docs:doctor`

### Snapshots

- `snapshot:review`
- `snapshot:update`

### Autofix

- `fix:all`
- `fix:clippy`
- `fix:format`
